The current version of this app, called WiSense, is being distributed for use at no cost to the end user for non-commercial use only, by the WiNGS laboratory at UW-Madison. The app can be used to collect spectral samples from the WiFi bands and to analyze these samples to determine which non-WiFi devices (Bluetooth, ZigBee, cordless phones, microwave ovens) are operating in the vicinity. The app also collects infrequent location information during its operation --- this allows us to analyze spatial behavior of spectral activity in the WiFi bands.


Data collection

In its default operation, the software will collect and send the following information to a server operated by the WiNGS laboratory. This information includes: (i) the spectral samples and optionally (ii) infrequent location information. Collection of location data can be disabled by the user. No personally identifiable information (such as IMEI number or MAC address) is collected. Finally, the user can access their own uploaded data from our server.


User controls

The app provides users with multiple settings through which they can control the above behavior. The app can be configured to determine location using WiFi or cellular signals only, which is known to be less accurate than using GPS. The app can also be configured to completely disable collection of location information. Finally, the app provides an option through which no data is uploaded to the server.


Data plans

All data upload to the server is done only over WiFi. Even if the device has a 3G/4G data plan, the collected data is never uploaded using such a plan. Hence, the user needs to keep WiFi on.


Research use of data

The data being collected in the server is going to be used for research purposes to better understand and analyze spectral behavior. The location information collected will allow researchers to understand spatial behavior of wireless interference patterns across bands. The data collected does not carry any personalized information of users, and hence no such analysis of personalized behavior is going to be attempted on this data. The spectral sample data collected may be published by the WiNGS laboratory team through a public data repository. The actual location coordinates of any individual device collected through this app will never to put in the public domain.
